# TSL-Translation-System
เอกสารเกี่ยวข้องการทำโปรเจกต์ระบบแปลภาษามือ
สรุปภาพรวมโครงการ
โครงงานนี้มีวัตถุประสงค์เพื่อสร้างเครื่องมือช่วยสื่อสารระหว่าง ผู้พิการทางการได้ยิน และ คนปกติ โดยการเปลี่ยนท่าทางภาษามือให้เป็นข้อความหรือเสียงโดยอัตโนมัติ

จุดเด่นและขั้นตอนการทำงาน
เทคโนโลยีหลัก: ใช้การประมวลผลภาพ (Image Processing) ร่วมกับโปรแกรม Python และไลบรารี MediaPipe เพื่อตรวจจับจุดสำคัญบนฝ่ามือ (Hand Landmarks)

การประมวลผล: 
1.  รับภาพผ่านกล้อง Web Cam แบบ Real-time
2.  ตรวจจับการเคลื่อนไหวและตำแหน่งของนิ้วมือ
3.  นำข้อมูลไปเปรียบเทียบกับฐานข้อมูลเพื่อแปลเป็นตัวอักษรหรือคำศัพท์

เป้าหมาย: ลดช่องว่างทางการสื่อสาร เพิ่มโอกาสในด้านการศึกษาและการทำงานให้กับผู้พิการ

ผลที่คาดว่าจะได้รับ:

1.  ระบบสามารถแปลภาษามือพื้นฐานได้อย่างแม่นยำ
2.  ผู้ใช้งานทั่วไปสามารถเข้าใจสิ่งที่ผู้พิการทางการได้ยินต้องการสื่อสารได้ทันทีผ่านหน้าจอ
